# 色塊選擇器 (Color Block Selector) 使用指南

## 新增功能

在 `ccm_test_3.py` 的視窗下方新增了**色塊編號輸入框**，用於快速選擇和操作特定的色塊。

## 功能說明

### 1. 色塊輸入框位置
- 位於窗口底部
- 標籤：「Color Block (1-24):」
- 輸入範圍：1-24（對應24色色卡）

### 2. 操作方式

#### 方法一：使用 Select 按鈕
1. 在輸入框中輸入色塊編號（1-24）
2. 點擊「Select」按鈕
3. 對應的色塊邊框會變成**紅色**並加粗，表示已選中

#### 方法二：鍵盤操作
1. **點擊輸入框**激活編輯模式
   - 輸入框邊框會變成**藍色**
2. **輸入編號**
   - 支援 0-9 數字輸入
   - 最多輸入2位數字（1-24）
3. **確認輸入**
   - 按 Enter 鍵確認並選擇色塊
   - 按 Escape 鍵取消編輯

### 3. 色塊調整

選中的色塊（紅色邊框）支援以下操作：

#### 移動方框
- **滑鼠拖拽**：將選中的方框移動到新位置

#### 調整大小
- **Shift + 滑鼠拖拽**：改變方框的寬度和高度

#### 顏色提取
- 選中色塊後，系統會自動從方框內提取平均顏色
- 該顏色的 Lab 值會即時顯示在右側的分析窗口

### 4. 應用場景

#### 用戶圖片模式（手動調整）
- 打開自己的圖片後（File > Open Image）
- 使用色塊輸入框快速定位到需要調整的色塊
- 通過拖拽調整方框位置和大小
- 系統自動計算該區域的顏色並應用 CCM 修正

#### 24色色卡模式
- 即使在標準24色色卡模式下也可使用此功能
- 用於快速導航和查看特定色塊的 Lab 值

### 5. 限制與提示

✅ **可以做**
- 輸入 1-24 的任意數字
- 在手動模式中調整選中色塊的位置和大小
- 根據色塊號快速定位特定顏色

❌ **不能做**
- 輸入超出 1-24 範圍的數字（會提示警告）
- 輸入非數字字符（只接受 0-9）

### 6. 界面元素說明

| 元素 | 說明 |
|------|------|
| 標籤 | "Color Block (1-24):" |
| 輸入框 | 黃色背景，顯示當前選擇的色塊編號 |
| Select 按鈕 | 點擊以確認並選擇指定的色塊 |
| 選中效果 | 選中的色塊邊框變為紅色，線寬加粗 |

### 7. 與其他功能的協作

- **Close Image File**：關閉用戶圖片後，輸入框保持可用，但色塊選擇只在手動模式生效
- **CCM 調整**：選中色塊並調整方框後，系統自動重新計算該色塊區域的 Lab 值
- **Reset 按鈕**：重置 CCM 矩陣不影響色塊選擇狀態

---

**提示**: 在手動模式中，通過組合使用色塊輸入框和方框拖拽功能，可以精確控制色卡的位置，從而實現更精準的色彩校正。
