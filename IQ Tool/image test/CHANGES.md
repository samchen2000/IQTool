# CCM Test 2 更新說明

## 新增功能

### 1. 每個滑桿下方增加增加/減少按鍵
- 在每個滑桿（共 9 個）下方添加了「-」（減少）和「+」（增加）按鍵
- 每次點擊增加或減少 0.05 的數值
- 數值受滑桿範圍限制，不會超出最小/最大值

### 2. 按鍵中間增加輸入框
- 在「-」和「+」按鍵之間添加了單行輸入框
- 輸入框實時顯示對應滑桿的數值
- 用戶可以直接在輸入框中編輯數值

### 3. 輸入框交互功能
- **點擊**輸入框時，邊框顏色變為藍色，進入編輯模式
- **鍵盤輸入**：支持數字、小數點和正負號
- **Backspace**：刪除最後一個字符
- **Enter**：確認輸入並更新滑桿
- **Escape**：取消編輸入框顏色恢復為黃色

## 技術細節

### 修改內容
1. **GUI 佈局調整**
   - 增加圖表高度：16x8 → 16x10
   - 增加底部空間：0.50 → 0.60
   - 統計資訊框向上移動（Y 座標 0.42 → 0.50）

2. **新增全局變數**
   - `buttons_inc`：增加按鍵列表
   - `buttons_dec`：減少按鍵列表
   - `input_boxes`：輸入框資訊字典列表
   - `slider_ranges`：每個滑桿的最小/最大值
   - `input_editing_data`：正在編輯的輸入框資訊

3. **新增函數和事件處理**
   - `create_button_callbacks()`：建立增加/減少按鍵回調
   - `on_pick_event()`：處理點擊輸入框事件
   - `on_key_event()`：處理鍵盤輸入事件

4. **更新 update() 函數**
   - 增加輸入框實時更新邏輯

## 使用方式

1. **使用滑桿**：直接拖動滑桿調整 CCM 矩陣參數

2. **使用增加/減少按鍵**：
   - 點擊「-」按鍵減少 0.05
   - 點擊「+」按鍵增加 0.05

3. **直接輸入數值**：
   - 點擊輸入框（邊框變藍）
   - 輸入新數值
   - 按 Enter 確認或 Escape 取消

4. **重置為單位矩陣**：點擊「Reset Identity」按鍵

## 顏色代碼
- **滑桿背景**：淡金黃色
- **減少按鍵**：淺紅色（懸停時變紅）
- **增加按鍵**：淺綠色（懸停時變深綠）
- **輸入框**：淺黃色（編輯時變淡青色）
