## Recalbox
- Recalbox 是一個全方位的復古遊戲系統，旨在讓使用者重新遊玩童年時期的所有家用主機、大型機台及電腦遊戲
。目前它支援超過 100 種以上的相容系統，涵蓋了各個世代的硬體
### 以下根據來源內容，將支援的系統分類整理如下：
## 1. 家用遊戲主機 (Home Consoles)
- Nintendo (任天堂): 
NESSuper Nintendo (超任)、Nintendo 64、Gamecube、Wii。
- Sega (世嘉): SG-1000、Master System、Megadrive、Mega-CD、Megadrive 32X、Saturn (土星)、Dreamcast。
- Sony (索尼): Playstation (PS1)、Playstation 2。
- Atari (雅達利): Atari 2600、Atari 5200、Atari 7800、Jaguar。
- 其他經典主機: PC Engine、PC Engine CD-Rom、PC Engine SuperGrafx、Neo-Geo CD、3DO、Intellivision
、Vectrex、Videopac G7000、Super Cassette Vision、PC-FX、Channel F、GX4000。
## 2. 掌上型遊戲機 (Handhelds)
- Nintendo: Game Boy、Game Boy Color、Game Boy Advance、Nintendo DS、Virtual Boy、Game & Watch、Pokémon mini。
- Sony: PSP。
- 其他掌機: Game Gear、WonderSwan、WonderSwan Color、Lynx、Neo-Geo Pocket、Neo-Geo Pocket Color、Watara Supervision
## 3. 電腦系統 (Computers)
- 經典電腦: Amiga (含 AGA、CDTV、CD32)、MSX /MSX2 / MSX Turbo R、Commodore 64、Vic-20、Apple II / IIGS、ZX Spectrum、ZX81、Atari ST / Atari 8bit、X68000、PC-9801、PC Dos、PC Windows。
- 其他系統: CPC、Thomson MO/TO、BBC Micro、Oric 1 / Atmos、SAM Coupé、Sharp X1、TRS-80 Color Computer、TI-99/4A、BK。
## 4. 大型機台、擴充系統與引擎 (Arcade & Engines)
- 大型機台: Mame、Neo-Geo、Naomi、Naomi GD-ROM、Atomiswave、Model 3、Daphne。
- 擴充與虛擬引擎: ScummVM、OpenBOR、EasyRPG、Uzebox、Family Computer Disk System、Satellaview、Nintendo 64DD、Palm OS、CD-i。  
``` 
除了上述系統外，Recalbox 還提供許多現代化的功能來提升復古遊戲體驗，例如即時存檔 (Instant save)、倒帶 (Rewind)、自動抓取遊戲資訊 (Scraping) 以及各式濾鏡與遮罩效果。
```
## 參數進入
- SSH 登入資訊如下：  
帳號 (Username): root  
密碼 (Password): recalboxroot

- Web 管理界面（推薦）： 在電腦瀏覽器輸入
http://recalbox.local/ （Windows 有時直接用 http://recalbox/）

- 預設名稱： 系統預設的 Hostname 為 'RECALBOX'
```
 一旦您修改了主機名稱，原本預設的網址
 （如 http://recalbox/ 或 http://recalbox.local/）將會失效。
 您必須改用新的名稱來存取（例如：http://[新名稱].local/）。
```
