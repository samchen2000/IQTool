## 修復的問題

### 問題 1：AttributeError 錯誤
**錯誤信息**：`AttributeError: 'FigureCanvasTkAgg' object has no attribute 'mpl_cid_key_press'`

**原因**：在第1215行嘗試斷開一個不存在的連接 ID
```python
fig.canvas.mpl_disconnect(fig.canvas.mpl_cid_key_press)  # 錯誤 - 該屬性不存在
```

**修復方案**：
- 移除了無效的 `mpl_disconnect()` 調用
- 直接連接整合的鍵盤事件處理器
- 新的鍵盤事件處理器能同時處理色塊輸入和原有的功能

**修改位置**：第1205-1217行

### 問題 2：方框無法使用拖曳方式調整大小
**原因**：
1. `on_mouse_press` 只是記錄起始位置，但沒有設置 `selected_box_idx`
2. `on_mouse_motion` 需要依賴 `selected_box_idx` 不為 None 才能工作
3. Shift+拖拽 調整大小功能因為沒有 `selected_box_idx` 而無法觸發

**修復方案**：
改進 `on_mouse_press()` 函數，使其：
1. 檢查滑鼠點擊是否在某個方框內
2. 正確設置 `selected_box_idx` 
3. 更新方框顏色反饋（點擊的方框變紅色）

**修改位置**：第759-793行
